# AnalChem Pre-Exam

A practice pre-examination designed for **Analytical Chemistry**, focusing on **Titration** and **Gravimetry** concepts. This project provides students with an interactive way to review and test their knowledge before the actual exam.

---

## 📖 Description
This repository contains an HTML-based quiz application for analytical chemistry. It is intended as a study aid for students preparing for pre-examinations, specifically covering **titration methods** and **gravimetric analysis**.

---

## 📝 Instructions
- The exam is composed of:
  - **Multiple Choice Questions (MCQ)**
  - **True or False**
  - **Identification**
- Students answer all questions within the browser.
- After submission, the system will automatically calculate and display the score.

---

## ✨ Features
- Interactive quiz interface
- Covers **titration** and **gravimetry** topics
- Multiple question formats (MCQ, True/False, Identification)
- Instant scoring feedback after submission
- Easy to expand with additional questions
